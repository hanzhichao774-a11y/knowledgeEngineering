## 一、Demo 演示脚本（用户视角）

```
用户上传一份企业制度文档（PDF/Word）
        ↓
管理智能体接收任务，判断应委派给"知识工程数字员工"
        ↓
知识工程数字员工开始执行：
  Step 1: 文档解析（多模态输入处理）
  Step 2: 本体提取（概念、术语、规则、关系）
  Step 3: Schema 构建（结构化输出）
  Step 4: 写入图数据库
  Step 5: 生成知识图谱
        ↓
执行过程中前端实时展示：
  - 当前执行的智能体身份
  - Token 消耗（输入/输出）
  - 调用的技能类型
  - 每步进度和中间产物
        ↓
完成后：
  - 可查看 Schema 内容（行列式浏览）
  - 可查看简化版知识图谱（前端渲染）
  - 可进入问答智能体（回答 + 归因追溯）
```

## 二、功能模块拆解

| 模块 | 说明 | 优先级 | 负责方 |
|------|------|--------|--------|
| **F1 任务输入** | 上传文档 / 输入自然语言任务描述 | P0 | 前端 |
| **F2 管理智能体委派** | 接收任务 → 判断工作线 → 委派数字员工 | P0 | 后端 (KodaX) |
| **F3 执行过程实时展示** | 左侧任务列表 + 右侧执行流（当前步骤、Token、技能） | P0 | 前端 |
| **F4 中间产物动态渲染** | 根据输出类型自动适配展示形式 | P0 | 前端 |
| **F5 文档解析** | 多模态输入处理（PDF/Word/Excel/图像） | P0 | 后端 (KodaX Skill) |
| **F6 本体提取** | 从文档中提取概念、术语、规则、关系 | P0 | 后端 (KodaX Skill) |
| **F7 Schema 构建** | 结构化输出，可浏览 | P1 | 后端 + 前端 |
| **F8 图数据库写入** | 将提取结果写入图数据库 | P1 | 后端 (KodaX Tool) |
| **F9 知识图谱渲染** | 前端简化版图谱可视化 | P1 | 前端 |
| **F10 问答 + 归因** | 基于知识图谱的问答，答案来源可查 | P2 | 后端 + 前端 |
